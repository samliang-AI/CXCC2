#### 数据质量标签页
- **完整性概览**: 
  - 有效记录数 (98%)

- **缺失字段详情**:

  - 缺失百分比
  - 严重程度 (高/中/低)

- **数据质量建议**:
  - 整体评价
  - 改进建议

## 💡 实现细节

### 文件验证函数
```typescript
const validateFile = (file: File): boolean => {
  // 1. 检查文件类型
  const allowedTypes = [...]
  if (!allowedTypes.includes(file.type) && 
      !file.name.match(/\.(xlsx|xls|csv|json)$/i)) {
    alert('不支持的文件格式！')
    return false
  }
  
  // 2. 检查文件大小
  const maxSize = 50 * 1024 * 1024
  if (file.size > maxSize) {
    alert('文件过大！')
    return false
  }
  
  // 3. 检查文件名
  if (file.name.length > 100) {
    alert('文件名过长！')
    return false
  }
  
  return true
}
```
